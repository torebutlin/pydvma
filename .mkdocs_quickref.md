# MkDocs Quick Reference

## Common Commands

```bash
# Install dependencies
pip install -r requirements-docs.txt

# Serve locally (with auto-reload)
mkdocs serve

# Build static site
mkdocs build

# Build with strict mode (fail on warnings)
mkdocs build --strict

# Deploy to GitHub Pages manually
mkdocs gh-deploy

# Clean build artifacts
rm -rf site/
```
